# Captain Tsubasa - New Era

Proyecto Unity: un juego de fútbol con poderes.

## Requisitos
- Unity 2020.3 LTS o superior (ajusta según tu versión).
- Git (se recomienda usar Git LFS para assets grandes).

## Cómo abrir
1. Clona el repositorio.
2. Abre la carpeta del proyecto en Unity Hub.

## Estructura recomendada
- Assets/ — recursos y scripts de Unity
- ProjectSettings/ — ajustes del proyecto
- Packages/ — dependencias Unity

## Notas
- He añadido un .gitignore para excluir carpetas generadas por Unity.
- Renombra/organiza los assets si deseas (p. ej. Assets/Art).
